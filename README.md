# Windows Security Investigations

This repository contains Windows Event Log investigations performed as part of my cybersecurity training and SOC analyst development.

Each investigation includes:
- Event details and context
- MITRE ATT&CK mapping
- Detection logic (KQL / SIEM queries)
- Threat hunting questions
- Screenshots
- Final conclusion and risk rating
